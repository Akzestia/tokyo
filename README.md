### Make your c++ code look nice ^~^

```c++
// Without tokyo ~_~

auto main(int argc, char* argv[]) -> int {
    // Do smth
}
```

```c++
// With tokyo ^~^
#include <tokyo/tokyo_main.hxx>

tokyo {
    // Do smth
}
```

### Install

Clone repo

```sh
gh repo clone Akzestia/tokyo
cd tokyo
```

Build 

```sh
just
# Or with demo
just xpp ON && just b
```

Install globally

```sh
just i
```
